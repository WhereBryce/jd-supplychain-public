JD 库存报告字段词典

京东商智库存报告（黄金眼 / 商智「库存明细下载」）明细字段的逐条识别词典：商品属性、销量 band、库存件数（现货 / 可用 / 可订购 及公式）、周转 / 滞销 / 不动销 / 库龄、采购与到货、出库与销量口径、有货天数 / 可售 / PV 现货率。文末附库存相关字段区分对比表。字段定义均取自官方《商智库存报告字段识别》。
实时过滤下方所有字段行；清空恢复全部。

字段说明

上下柜状态商品当前的上下柜状态，指主站场的上下柜状态
中台上下柜状态中台上下柜状态，指商品的上下柜状态，包含下柜、可上柜、上柜三种状态
上架时间即 SKU 的创建时间
下柜时间下柜时间，最近一次下柜时间

首次上柜时间首次上柜时间

SPU 编码进入商品详情页默认的第一个 sku 编码，指的是 SPU，不是影分身商品的主品

长度 / 宽度 / 高度商品长度 / 宽度 / 高度（mm）

冷链标记分为控温（10 -18℃）、冷藏（0-4℃）、冷冻（零下 18℃ 以下）、深冷（零下 30℃）、鲜活、产地直送、鲜品（冷藏存常温配）、冷藏(2-8度)、冷冻(-15- -25℃)
是否新品按照首次入仓 30 天判断的

二销量分级 Band

字段说明
全国 7 日三级销量 band根据近 7 天的销量对这个类目中的所有 sku 进行排序分级，排在前 20% 的为 A，20%-40% 为 B，以此类推，分为 ABCDEF 六个 band
全国 28 日三级销量 band根据近 28 天的销量对这个类目中的所有 sku 进行排序分级，排在前 20% 的为 A，20%-40% 为 B，以此类推，分为 ABCDEF 六个 band

三库存件数与可用 / 可订购

字段说明
现货库存在京东仓库的现货库存件数
内配在途库存内配途中数量，记在目的仓下
库存件数现货库存件数 + 内配在途件数
内配入内配计划入，源仓还未完成出库的数量
内配出内配计划出，还未完成出库的数量
已锁定已锁定库存，包括订单取消锁定、供货商退货锁定、POP 团购等业务上需要锁定库存的数量，不允许用户下单
可用库存前台可以销售的库存数量，现货 + 内配在途 + VMI 库存 - 锁定 - 内配出 - 不可售 - 订单预定（订单预占未出库的商品数量）。VMI 库存：VMI 是一种供应链模式，指的是将 POP 入仓的库存共享给自营，VMI 库存指的是共享给自营的库存件数
可订购库存代表可以订购的库存，= 现货 + 内配在途 + VMI 库存 - 锁定 + 内配入 - 内配出 - 不可售 - 订单预定 + 采购未到货。与可用库存关系：可订购库存 = 可用库存 + 采购未到货 + 内配入
订单预定用户下订单后未出库的订单对应的件数，包含下单未付款 + 付款未发货两部分
VMI 库存VMI 是一种供应链模式，指的是将 POP 入仓的库存共享给自营，VMI 库存指的是共享给自营的库存件数
不可售残品、样品、报废品等
现货库存成本现货库存数量 * 仓报价
可用库存金额可用库存 * 仓报价
退供应商在途未完成的退货单对应的退货数量
预定状态指没有现货时，但为保持商品有持续的展示和销量，通过采销或者系统设置商品状态（前台显示预定）
预定数量指没有现货时，但为保持商品有持续的展示和销量，通过采销或者系统设置商品状态（前台显示预定）
无货下单包含两种订单类型：1）企业大客户的无货下单；2）家电专卖店的免限订单
大件待预占数量客户下单已付款且符合破界释放中订单明细（目前可订购库存中未扣减大件 X-X 天待预占数量的数据）

四周转 / 滞销 / 不动销 / 库龄

字段说明
数量周转库存件数 / 日均出库销量
滞销件数总库存数量 - 滞销天数阈值（指周转超多少天为滞销）* 日均出库销量；若日均销量为 0，则滞销库存数量 = 总库存数量 - 无销售基本备货数量。滞销库存中剔除新品，新品首次入仓天定义需查看规则配置（默认 30 天）。滞销天数阈值、参考的日均出库销量、无销售基本备货数量、新品判断规则为采销配置的口径。中小件、生鲜按照 RDC 维度统计，大件按照中心仓 + 卫星仓维度，其他按照配送中心维度统计
滞销库存金额滞销件数 * 仓报价
不动销库存件数连续 XX 天以上有货且 XX 天销量为 0 的 SKU 的库存件数 - 无销售基本备货。连续 XX 天以上有货、XX 天销量为 0 和无销售基本备货数量为采销配置的口径。中小件、生鲜按照 RDC 维度统计，大件按照中心仓 + 卫星仓维度，其他按照配送中心维度统计
不动销库存金额不动销库存件数 * 仓报价
长库龄库存件数按照统计规则部门设置的库龄阈值计算超出阈值的库存数量，指在库时间超多少天为长库龄
长库龄库存金额长库龄库存件数 * 仓报价
X-X 天库存数量库龄在 X 天（时间范围内）的库存数量。每批次库存的库龄 = 当前日期 - 该批次首次采购入库日期。内配不会影响库龄的计算。库龄库存 = 逻辑批次口径的现货库存 + 内配在途库存
X-X 天库存金额X-X 天库存数量 * 仓报价

五采购与到货

字段说明
采购未到货包含订单提交后到完成之前的有效采购单的库存件数，已回告取回告数量，没有回告取原始下单数量
采购未到货 - 已回告已回告的采购未到货件数
采购未到货 - 已预约已预约送货的采购未到货件数
采购未到货金额= 昨日采购未到货数量 * 下单价
采购未到货金额（已回告）= 昨日采购未到货数量（已回告）* 下单价
采购未到货金额（已预约）= 昨日采购未到货数量（已预约）* 下单价
采购入库件数该时间段内京东自营仓库验收入库的商品数量
采购入库金额该时间段内京东自营仓库验收入库的商品采购金额
采购下单件数该时间段内有效状态下的采购单初始的下单件数
退供应商件数该时间段内提货的有效退货单的库存件数

六出库与销量口径

字段说明
出库商品件数按出库地统计近 X 日出库销量，取库房出库打包的数量，不减退货、拒收。只包含自营仓库出库的订单，不包含厂直、VMI 等订单。出库 + 厂直 + VMI 出库则为自营总体的出库件数。
厂直出库商品件数厂直订单对应的出库件数
VMI 出库商品件数VMI 订单对应的出库件数。VMI 是一种供应链模式，指的是将 POP 入仓的库存共享给自营。
收货地商品件数按照订单收货地址对应到配送中心，统计的收货地销量，会减去退货、拒收、取消的订单

七有货天数 / 可售 / 现货率

字段说明
xx 日有货天数近 X 日可用库存 > 0 的天数，根据下载数据的统计维度计算，比如 RDC 维度是按照 RDC 汇总的可用库存判断的有货天数，配送中心维度是按照配送中心自己的可用库存判断的有货天数
预计可售天数对于库存可以卖多少天的预估。预计可售天数 = 可用库存 /（近 28 日收货地商品件数 / 28 日有货天数），若 28 日有效销量或 28 日有货天数 = 0（或空值），则可售天数 = 可用库存 / 0.1
PV 现货率该时间段内有货的 SKU 的浏览量 / 总的 SKU 的浏览量，即有货 PV / 总 PV，剔除虚拟品类、虚拟组套、影分身商品、售完即止的 SKU
